# Market scan template

## Objetivo

Documentar un **barrido acotado** que se pueda ejecutar en tiempo limitado y devolver **decisiones o preguntas** claras.

## Plantilla

```markdown
# Market scan — [tema / género] — [fecha]

## Scope
- Pregunta principal:
- Incluye: …
- Excluye: …
- Región / plataforma / audiencia (si aplica):

## Fuentes (tipo, no necesariamente URL exhaustiva)
- Reviews agregados / crítica
- Foros / comunidades (muestra)
- Datos públicos (rankings, fechas de lanzamiento)
- Gameplay público (trailers, longplays)

## Hallazgos por bloque
### Demanda / volumen (cualitativo o con fuente)
### Features dominantes percibidas
### Saturación / clichés
### Huecos / “underserved”

## Riesgos y sesgos
- …

## Preguntas abiertas para PO
- …

## Próximo paso sugerido
- benchmark-summary / feature-opportunity-analysis / cerrar
```

## Reglas

- Marcar **inferencia** vs **dato citado**.
- No expandir scope mid-scan sin anotarlo.
